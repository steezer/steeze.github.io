### 1. PHP版本及扩展要求
- PHP >= 5.4
- PHP PDO 扩展
- PHP Mbstring 扩展
- PHP XML 扩展

### 2. 支持的服务器和数据库环境
 - 支持Windows/Unix服务器环境
 - 可运行于包括Apache、IIS和nginx在内的多种WEB服务器和模式
 - 支持swoole服务器模式运行，安装[swoole](http://www.swoole.com/)扩展后可直接运行
 - 支持Mysql、MsSQL、PgSQL、Sqlite、Oracle、Firebird、Mongo以及PDO等多种数据库和连接
